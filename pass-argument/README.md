# Passing Argument in Shellscript

## Using Space

Sample Command:
```console
$ ./by-space-key.sh -e conf -s /etc -l /usr/lib /etc/hosts
```

## Using Equal Sign

Sample Command:
```console
$ ./by-equal-sign.sh -e=conf -s=/etc -l=/usr/lib /etc/hosts
```
